# Monte Carlo Options Pricer

Interactive Monte Carlo simulator for option pricing with real-time visualizations.

## Features

- **Monte Carlo simulation** for European options (calls/puts)
- **Interactive parameter controls**: volatility, drift, time horizon, strike
- **Visual outputs**: price paths, distribution histograms, payoff heatmaps
- **Risk metrics**: Greeks (delta, gamma, vega, theta), VaR, expected shortfall

## Stack

- Python (NumPy, SciPy, Matplotlib)
- Optional: Next.js/T3 frontend with interactive sliders

## Roadmap

- [ ] Basic Black-Scholes-Merton Monte Carlo engine
- [ ] CLI with parameter inputs and plot outputs

Quant practice project for building intuition around stochastic processes, risk-neutral pricing, and computational finance.
